codeBMB is an NSF supported initiative to prepare faculty members to teach students the computational skills they need to join the workforce, ready to take on the next generation of scientific challenges. This webinar is among the first in a series of workshop and presentations to support coding in biochemistry and development of tools to teach coding in biochemistry and molecular biology.
